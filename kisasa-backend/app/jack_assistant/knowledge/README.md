# Jack Knowledge Documents

Add Jack's farm knowledge files in this folder as `.md` or `.txt`.

Good document format:

- Start with a clear heading, for example `# Tomato Early Blight`.
- Include symptoms, causes, immediate action, treatment, prevention, and safety notes.
- Use practical farmer language.
- Mention when the farmer should contact an extension officer, agrovet, or verified expert.

Jack searches these files whenever someone asks him a chat question or tags `@Jack` in a comment.
